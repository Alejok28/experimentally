# RecommendedItem
RecommendedItem is a component that shows a the thumbnail of the video and a tooltip to see the title. This component is in charge to select a video to play in the VideoPlayer component.

## Properties

| Name          | Type     | Required     |Description                                |
|:--------------|:--------:|:------------:|:------------------------------------------|
| video         | Object   | true         | Object with the info commig from YoutubeApi|
| handleClick   | Func     | true         | This function is in charge for selecting a video to play in the VideoPlayer component |
